# Premier-style Hospital Readmission Project

This is a fast portfolio project designed to match the kind of healthcare analytics Premier asked for:
- healthcare data analytics
- relational databases
- Python
- model research / development support

## What to download
Use the UCI dataset:
**Diabetes 130-US Hospitals for Years 1999-2008**

Main file needed:
- `diabetic_data.csv`

## Fast workflow
1. Download `diabetic_data.csv`
2. Put it next to `premier_readmission_project_starter.py`
3. Run the script
4. Use the output charts and CSVs in a portfolio folder
5. Put the resume bullets below into your resume or LinkedIn project section

## Resume bullets
- Built a healthcare readmission risk project using a public U.S. hospital dataset with 100k+ encounters
- Cleaned and prepared clinical data for modeling, including missing-value handling, feature reduction, and target engineering
- Analyzed 30-day readmission patterns by age band and diagnosis grouping
- Trained a logistic regression model to estimate readmission risk and exported top predictive factors

## Suggested project title
**Hospital Readmission Risk Modeling Using U.S. Healthcare Data**
