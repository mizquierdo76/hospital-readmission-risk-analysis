Hospital Readmission Risk Analysis¶

This project analyzes U.S. hospital patient data to identify factors associated with 30-day readmissions. Using Python and healthcare datasets, the analysis explores demographic patterns, diagnoses, and patient utilization factors that influence readmission risk. A logistic regression model is used to predict which patients are most likely to be readmitted.

Hospital Readmission Risk Analysis¶

Overview¶

This project analyzes U.S. hospital patient data to identify factors associated with 30-day readmissions.

Objectives¶

  • Explore readmission patterns across patient groups
  • Identify high-risk diagnoses and demographics
  • Build a simple predictive model for readmission risk

Tools Used¶

  • Python
  • Pandas
  • Matplotlib
  • Scikit-learn

Business Value¶

Reducing avoidable readmissions can improve patient outcomes and lower hospital costs. This analysis helps identify patient groups that may benefit from earlier intervention and follow-up care.
